## 스터디 진행 방법

- 전 주에 해당 범위 중 발표할 파트를 분배한다.(김세원과 이진우 2강, 교수님 1강)
- 일주일에 5강씩 다 같이 수강한다.
- 매주 자신이 맡은 부분을 markdown 형식으로 정리하여 GITHUB에 Pull Request한다.
- 모든 참가자는 해당 주에 다른 참가자들이 올린 자료를 정독하고 참가한다.
- 모임에서 자신이 공부한 부분에 대해서 10~15분 동안 발표를 진행한다.
- 발표를 마치고 질의응답 시간을 갖는다.
