- 전 주에 해당 범위 중 발표할 파트를 분배한다.(김세원과 이진우 2강, 교수님 1강)

- 모임에서 자신이 공부한 부분에 대해서 10~15분 동안 발표를 진행한다.
- 발표를 마치고 질의응답 시간을 갖는다.
