# 06 Topic

- Topic : Node들 사이에 데이터가 오가는 길의 이름
- Publiher : topic을 보내는 주체
- Subscriber : topic을 받는 주체
- Topic은 1:N 통신이다.
- Topic은 기본적으로 지속적으로 데이터를 보낸다.
